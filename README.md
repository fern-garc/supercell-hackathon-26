A 3D horror experience built with Three.js + Gemini. Navigate a procedurally generated maze, dodge the creature lurking in the shadows, and find the ancient ritual book to escape!

## üöÄ Getting Started

To run the game locally, you can use a simple Python HTTP server:

1. Open your terminal in the project directory.
2. Run the following command:
   ```bash
   python -m http.server 8000
   ```
3. Open your browser and navigate to `http://localhost:8000`.

## üìú Goal

* **The Mission:** You are trapped in the Maze of Monster.
* **The Key:** Find the **Ritual Book** hidden somewhere within the shifting walls.
* **The Threat:** A relentless monster hunts you. Use your flashlight wisely and save your Crystal Flash for emergencies.
